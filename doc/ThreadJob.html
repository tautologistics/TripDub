Call

<static> Object Call(<nsIThread> thread, <Boolean> async, <Function> func )
    Calls the passed function on another thread (a)synchronously
    Parameters:
      thread - The thread to execute the code on
      async - A flag indicating whether to execute the function synchronously (False) or asynchronously (True)
      func - The function to execute
      arg1_N - Optional argument(s) to pass to the function being executed
    Returns:
      When the function is called synchronously the return value of the called function is returned
