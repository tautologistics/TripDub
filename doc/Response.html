Redirect

void Redirect(<String> location)
    Creates an HTTP redirect response (302) for the client. Calling this value modifies the Headers collection by adding/modifying/removing the "Location" header
    Parameters:
      location - The URL to redirect the client to. If this value is null then the redirect is canceled ("Location" header removed and StatusCode set to 200)
